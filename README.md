# trucklog
TruckLog – osobní aplikace pro evidenci jízd, tankování
## Funkce
- Evidence jízd a tankování / nabíjení.
- Evidence vozidel a profilů řidičů.
- Historie a export dat do CSV.
- Přihlášení a cloudová synchronizace přes Firebase.
- Role uživatelů (admin, dispečer, řidič) pro řízení oprávnění.
